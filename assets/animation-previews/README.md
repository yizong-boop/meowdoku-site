# Meowdoku 动效预览

这里是从 APK 中恢复的 Spine 4.2.43 动效生成的浏览器预览文件：

- `.webm`：资源站页面默认播放格式，体积较小，支持浏览器循环预览。
- `.gif`：便于下载、转发和在不支持 WebM 的工具中快速查看。
- `manifest.json`：预览文件、原始资源路径、选中的动画片段和该骨骼包含的全部动画名。

当前共生成 14 组预览，覆盖 41 条 Spine 动画资源。原始 APK 没有直接提供 GIF、WebM 或 MP4；这些文件是使用 Spine 4.2.43 runtime 根据 `.skel + .atlas + 纹理` 离线渲染得到的参考预览，不替换原始资源。

资源清单中的两条 Godot `GenericPopup*.res` 属于 Godot Animation 资源。它们需要放回对应 Godot 场景、节点和运行时状态后才能准确播放，因此没有伪造静态动画文件；资源站会保留原始资源并明确标注。
